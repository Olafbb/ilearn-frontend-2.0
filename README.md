# ilearn-frontend-2.0

Project created with Spring and vaadin frameworks connected with discord and dropbox API. To properly use
remember to have backend project launched.

Functions:

Simple view for student with most useful informations for him, connections with discord lobbies (per lesson),
uploading homeworks, chat, actual number of people who are in discord lobbies (having lessons).
